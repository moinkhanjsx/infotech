# How to Open the Website in Your Browser

There are several easy ways to open and view the website:

## Method 1: Direct File Opening (Easiest)

1. **Navigate to the project folder:**
   ```
   e:\react project\assignment\infotech
   ```

2. **Find the `index.html` file**

3. **Double-click on `index.html`**
   - This will open the website in your default browser

## Method 2: Right-Click Method

1. **Right-click on `index.html`** in File Explorer
2. Choose **"Open with"**
3. Select your preferred browser:
   - Google Chrome
   - Microsoft Edge
   - Firefox
   - Safari

## Method 3: Drag and Drop

1. **Open your browser** (Chrome, Edge, Firefox, etc.)
2. **Drag the `index.html` file** from File Explorer
3. **Drop it into the browser window**

## Method 4: Using VS Code (If you have Live Server extension)

1. **Right-click on `index.html`** in VS Code
2. Click **"Open with Live Server"**
3. This will open the website with live reloading

## Method 5: Direct URL in Browser

1. **Open your browser**
2. **Press `Ctrl + L`** to focus on the address bar
3. **Type or paste:**
   ```
   file:///e:/react%20project/assignment/infotech/index.html
   ```
4. **Press Enter**

## Method 6: VS Code Preview (Simple)

1. **Open `index.html`** in VS Code
2. **Right-click** inside the file
3. Select **"Open in Default Browser"** (if available)

## üîß Troubleshooting:

**If styles aren't loading:**
- Make sure `styles.css` is in the same folder as `index.html`
- Check that both files are in: `e:\react project\assignment\infotech`

**If fonts look different:**
- Make sure you have an internet connection (fonts load from Google Fonts)

**If the page looks broken:**
- Try a different browser
- Clear your browser cache (`Ctrl + Shift + Delete`)
- Make sure you're opening `index.html`, not any other file
